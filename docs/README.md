# Unit testing in C# [MSTest]
This is a simple example of how to write unit tests in C# using MSTest.


## Prerequisites
- [.NET Core SDK](https://dotnet.microsoft.com/download)
## Run the tests
To run the tests, execute the following command:
```bash
dotnet test
```
